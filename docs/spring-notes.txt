25-7-07
config    - 보안, CORS, JSON 등 전역 설정 클래스 모음
controller- HTTP 요청을 받아 응답을 반환하는 계층
domain    - DB 테이블 구조를 정의하는 @Entity 클래스
dto       - 요청·응답용 데이터 구조 (DB와 분리된 값 객체)
exception - 에러 종류 정의 및 예외 처리 핸들러 위치
repository- DB에서 데이터 조회·저장 쿼리(DB에 무엇을 달라고 요구하는 명령)를 정의
service   - 실제 로직과 흐름을 처리하는 비즈니스 계층
util      - 공통 도구성 클래스 (날짜, 문자열 등 재사용 기능)

FastAPI (Python) vs Spring (Java)
전자는 빠른 구축이 장점, 규모가 커질수록 높은 자유도로 인해 구조가 엉망이 되기 쉽다는게 단점

향후 대규모 서비스로의 확장을 대비해서 후자를 선택.
